=========
RunIt
=========
.. image:: src/runit.ico

Quick run window command liking window "run", but more features.

Compile
========
::

    mkdir build
    cd build
    # nmake
    cmake -G "NMake Makefiles" ../
    nmake

Run
======
::

    runit.exe

Usage
=======
Press default hot key ``insert`` to popup window.

Internal command::

    /about
    /config
    /edithosts
    /eject <disk>
    /lock
    /logoff
    /quit
    /restart
    /runas <application>
    /runscreensaver
    /shutdisplay
    /shutdown
    /syserr <error code>
    /url <link>

Input ``/config`` for more settings.
